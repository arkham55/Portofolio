# Development Roadmap & Sprint Schedule

This project is divided into ten distinct Sprints. Progress will be tracked against this master schedule.

| Sprint | Objective | Deliverables | Status |
| :--- | :--- | :--- | :--- |
| **Sprint 1** | **Planning** | Root repo templates, LICENSE, docs folder skeleton, initial planning schemas. | **Done** |
| **Sprint 2** | **Requirement Analysis** | Functional spec requirements, UI design rules, GitHub integration mapping. | Planned |
| **Sprint 3** | **Design System** | Tailwind configuration variables, dark/light scale settings, spacing guidelines, typography. | Planned |
| **Sprint 4** | **Architecture** | Next.js 15 initialization, TS path aliases, feature directories, static data layer configuration. | Planned |
| **Sprint 5** | **Foundation Components**| Shadcn UI configuration, Theme Provider, Navbar component, global layout structures. | Planned |
| **Sprint 6** | **Portfolio Sections** | Implementation of Hero, About, Education, Achievements, Skills, Learning, Contact, and Footer. | Planned |
| **Sprint 7** | **Project Showcase** | Detailed multi-dimensional project showcase blocks and sub-routes. | Planned |
| **Sprint 8** | **Interactive Experience**| Dynamic GitHub APIs integration, custom Framer Motion transitions. | Planned |
| **Sprint 9** | **Optimization & Testing**| Dynamic imports, lazy loading, image optimization, Lighthouse audits, validation. | Planned |
| **Sprint 10** | **Deployment** | Production build, sitemaps, robots.txt, metadata, Vercel deployments. | Planned |
